# 🎈 ㄅㄆㄇㄈ大挑戰 - 注音聽音射擊教育遊戲

一款結合**遊戲化學習（Gamification）**概念的注音符號辨識射擊遊戲。玩家需要仔細聆聽發音，並快速射擊對應的注音氣球。本專案支援**滑鼠**、**GUN4IR/OpenFIRE 光線槍**，以及**Webcam 鏡頭無硬體體感**三種操作方式，並提供單人挑戰、雙人合作與雙人對抗等模式，適合兒童或注音學習者在娛樂中訓練聽力與反應力。

---

## 🌟 核心特色

*   🏫 **黑板粉筆塗鴉風格**：介面採用溫馨擬真的深綠色黑板背景與粉筆質感字體，並具備豐富的動態粒子爆裂特效。
*   🗣️ **教育部官方語音授權**：音檔源自中華民國教育部《國語注音符號手冊》，朗讀發音精準標準。
*   🎮 **三種輸入模式隨選隨玩**：
    *   **1P 滑鼠模式**：使用滑鼠移動，左鍵射擊，相容性最高。
    *   **鏡頭體感模式**：免購買硬體！只需舉起手掌即可控制螢幕準星，在空中做出**「握拳」**動作即可射擊。
    *   **光線槍模式**：免去繁瑣的校正步驟，隨插即用（支援 GUN4IR、OpenFIRE 等相容 Gamepad API 的光槍設備，建議切換全螢幕）。
*   👥 **雙人遊玩支援 (2P)**：
    *   **2P 合作模式**：兩名玩家同屏擊碎氣球，共享分數與 Combo 加成，挑戰最高評分。
    *   **2P 對抗模式**：兩名玩家同屏競技搶答，擊中正確氣球即加分並切換下一題。
*   ⚙️ **個人化配置設定**：支援語音重播間隔調整、題目語音/背景音樂/遊戲音效獨立音量控制、氣球下降速度（慢速/標準/快速）以及光槍抗抖平滑度調整。

---

## 🛠️ 技術棧

*   **前端核心**：HTML5 Canvas、Vanilla JavaScript、Vanilla CSS (不使用第三方肥大框架，啟動極速)。
*   **體感識別**：Google MediaPipe Hands 機器學習手勢追蹤技術。
*   **離線模型優化**：本專案已將 MediaPipe 必要的 `.wasm`、`.tflite` 模型與 JS 庫本地化至 `js/lib/mediapipe/`，**無網路連接時依然能 100% 離線執行體感偵測**。
*   **硬體串接**：HTML5 Gamepad API (用於對接光線槍的搖桿與按鍵訊號)。

## 🚀 如何在本地運行

本專案支援**兩種本機執行方式**，程式會自動根據加載協定切換檔案來源，完美避開瀏覽器限制：

### 方式一：有網路時直接雙擊開玩（免安裝與啟動伺服器）
如果您的電腦目前處於連網狀態，可以直接**雙擊點開 `index.html`**。
*   **背後原理**：當偵測到使用 `file://` 協定開啟時，程式會自動將 MediaPipe 體感模型導向至公用 CDN 載入（該 CDN 具備 CORS `*` 許可標頭），因此您**不需要啟動任何本地伺服器**便能直接使用鏡頭體感功能！

### 方式二：完全離線運行（使用專案內建伺服器）
如果您處於無網路（離線）環境，或想極速載入：
1.  在專案根目錄按滑鼠右鍵，選擇「使用邏輯終端/PowerShell 開啟」。
2.  執行專案內建的伺服器啟動指令：
    ```powershell
    .\run_server.ps1
    ```
3.  伺服器將會在 `http://localhost:8080/` 啟動，並自動在瀏覽器中開啟。此時將完全載入本地的離線模型，支援 100% 離線遊玩！

## 🌐 部署至 Web 伺服器

本專案為純前端網頁應用，您可以直接將專案目錄下的所有檔案上傳至任何網頁伺服器（如 Apache、Nginx、IIS 或各式雲端靜態託管服務）進行部署。

### 📌 部署注意事項：
1. **無 CORS 跨來源問題**：當網頁程式與 MediaPipe 本地離線模型（位於 `js/lib/mediapipe/`）部署在同一個伺服器網域下時，瀏覽器會以同源（Same-Origin）方式載入，因此完全不會遇到 CORS 限制。
2. **必須使用 HTTPS 協定**：由於「鏡頭體感模式」需要存取 Webcam，瀏覽器安全規範規定**必須在 HTTPS 安全連線**（或本地 `http://localhost`）下，才允許網頁請求並啟用相機權限。請確保部署後的網站已啟動 SSL。

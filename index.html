遊戲說明

遊戲玩法

  • 遊戲開始後，系統會唸出一個注音符號（例如：「ㄅ」），請仔細聆聽。
  • 多個彩色注音氣球會從上方掉落，其中只有一個是正確答案。
  • 用滑鼠或光線槍射擊正確的注音氣球來得分並累積連擊 (Combo)！
  • 若擊中錯誤氣球，或漏看讓正確氣球掉出警戒線，則會扣除生命值或時間。

模式操作與快速鍵

  • 滑鼠模式：使用滑鼠移動瞄準，按滑鼠左鍵進行射擊。
  • 鏡頭體感模式：舉起單手手掌對準鏡頭，以「手掌中心」控制準星移動；在空中做出「抓取 (握拳)」的動作即可觸發射擊與按鈕點選，免硬體門檻且動感十足！
  • 光線槍模式：支援 GUN4IR、OpenFIRE 等相容 Gamepad API 模式的光線槍，隨插隨用、免手動校正（強烈建議使用右上角 全螢幕 按鈕以確保準心完美對齊）。
  • 鍵盤快速鍵：遊戲中按下 ESC 鍵可隨時暫停；按下 Space 空白鍵 可重聽發音。
